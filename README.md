## 🎬 Movie Recommendation System

An end-to-end **Machine Learning powered Movie Recommendation System** that recommends movies based on user preferences using **Collaborative Filtering**.

### ✨ Highlights

- 🎯 Personalized movie recommendations
- 🤖 Machine Learning with **Scikit-learn**
- 📊 Built using the **MovieLens Dataset**
- 🗄️ MySQL database hosted on **Railway**
- 🎥 TMDB API integration for posters & movie details
- 🌐 Deployed using **Streamlit Cloud**
- ⚡ Fast, clean and interactive web interface

### 📈 Dataset

- 🎬 **9,742 Movies**
- ⭐ **100,836 Ratings**
- 👥 **610 Users**

### 🛠 Tech Used

Python • Pandas • NumPy • Scikit-learn • MySQL • Streamlit • Railway • TMDB API • Git • GitHub
