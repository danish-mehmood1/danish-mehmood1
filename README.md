## 🎬 Featured Project

### Movie Recommendation System
**An end-to-end AI-powered recommendation web application**

An intelligent movie recommendation engine that uses collaborative filtering to provide personalized suggestions. Built with production-grade architecture and deployed live.

**Key Features:**
- 🔧 Collaborative filtering algorithm with Python & Scikit-learn
- 🗄️ MySQL database with Railway Cloud hosting
- 🎥 Real-time movie data integration via TMDB API
- ☁️ Fully deployed and accessible via Streamlit Cloud
- 📊 Trained on 9,742+ movies and 100,836+ user ratings

**Tech Stack:** Python | Pandas | NumPy | Scikit-learn | MySQL | Streamlit | Railway
